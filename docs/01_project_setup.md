# Task 01 — Project Setup

## What was done
- Created the project repository on GitHub
- Set up a Python virtual environment (venv) in PyCharm
- Created the folder structure:
  - `data/raw/` — for exported Anki files
  - `docs/` — for task documentation
- Added three Anki exports to `data/raw/`:
  - `200_words.txt` (general vocabulary)
  - `HSK2.txt` (HSK level 2 vocabulary)
  - `HSK3.txt` (HSK level 3 vocabulary)
- Created `.gitignore` to exclude virtual environment and IDE files

## Why
A clean project structure and version control from the start 
makes development easier to manage and review. Keeping raw 
data in a dedicated folder separates input from code.
