# Task 09 — Requirements and README

## What was done
- Generated `requirements.txt` using `pip freeze` to list all 
  project dependencies with exact versions
- Updated `README.md` with project description, tech stack, 
  setup instructions, and usage guide

## Why
A requirements file allows anyone to recreate the development 
environment and run the project. Clear setup instructions in 
the README make the project accessible and professional.
